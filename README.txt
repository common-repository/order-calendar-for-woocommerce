Show WooCommerce orders on a calendar

== Description ==
[Front end demo](https://woo-restaurant.com/order-calendar/)
<blockquote>
This is an addon plugin for <a href="https://www.piwebsolution.com/cart/?add-to-cart=621&variation_id=679">Order Delivery date and time pro plugin</a> or its free version <a href="https://wordpress.org/plugins/pi-woocommerce-order-date-time-and-type/"
>Order Delivery date and time free</a> (do not use this plugin with other date and time plugins)
</blockquote>

This plugin allows you to view Orders received on a calendar

* It will show the order received on a calendar
* Show the order count on each day on the calendar
* You can view number of delivery that you have to do on certain date on the calendar
* You can view number of Pickups that will be happening on certain date on the calendar


== Delivery and pickup dates on calendar =
To use this feature you will need to have the Delivery / Pickup date and time plugin installed on your site, you can use either the free version or the pro version of that plugin.

https://www.youtube.com/watch?v=z0BzI2Mc2cs

[Get Pro version of the Delivery date and time plugin](https://www.piwebsolution.com/cart/?add-to-cart=621&variation_id=679)
[Get free version of the Delivery date and time plugin](https://wordpress.org/plugins/pi-woocommerce-order-date-time-and-type/)

* It can show delivery date on the calendar
* Show total number of product ordered for a date
* Show total number of product ordered for a date organized in time-slot group
* Show pickup dates on the calendar
* It will show the order and pickup on a date along withe the time slot, so you can know how many order has to be completed in particular time slot
* This can be helpful for you to see upcoming delivery and pickup dates
* Show calendar on the front end using the short-code [order_calendar]
* Your staff can login into the front end of the site to see order detail
* Assign order detail access as per the employee role
* If an employee is in a production show them product detail of the order
* If an employee is in delivery then only show him customer detail or even product detail 
* It shows total quantity of product ordered per time slots of a particular date

== Frequently Asked Questions ==
= There is no calendar when I go to delivery and pickup date calendar =
It requires you to install the plugin Delivery / Pickup date and time plugin, that plugin will collect the delivery or pickup date when the order is placed and those dates will be shown here
you can use Free version or the pro version of the Delivery / Pickup date plugin for this purpose
[Get Pro version of the Delivery date and time plugin](https://www.piwebsolution.com/cart/?add-to-cart=621&variation_id=679)
[Get a free version of the Delivery date and time plugin](https://wordpress.org/plugins/pi-woocommerce-order-date-time-and-type/)

= I want to show my employee only customer detail in the order =
You can create a new user role in website say customer detail, and assign your all such employees in that particular group and then they will be able to see the customer detail only not the product that they order_calendar

= I only want to show the product ordered in order and not the customer detail =
Yes you can do so, assign all such employees to one group and assign product access to them and remove customer detail access 

= I want to see the order arranged in time slot group = 
In pro version when you click on particular date in (pickup or delivery) then it will show the orders arranged in a time slot 

= I will like to see, total quantity of different product order for a date =
Yes you can do so in the pro version

= Is it HPOS compatible =
Yes the Free version and PRO version both are HPOS compatible
